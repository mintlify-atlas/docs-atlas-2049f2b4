## Overview

Onchain Portfolio uses **RainbowKit** to provide a seamless wallet connection experience. The application is configured to work on the **Sepolia testnet**, allowing you to interact with smart contracts without using real ETH.

## RainbowKit Integration

The wallet connection functionality is powered by RainbowKit, which provides:

- Multiple wallet support (MetaMask, WalletConnect, Coinbase Wallet, etc.)
- Beautiful, customizable UI components
- Automatic network switching
- Account management

### Configuration

The RainbowKit setup is configured in `src/context/RainbowKitContext.js`:

```javascript src/context/RainbowKitContext.js
import { getDefaultConfig, RainbowKitProvider } from "@rainbow-me/rainbowkit";
import { WagmiProvider } from "wagmi";
import { sepolia } from "wagmi/chains";
import { QueryClientProvider, QueryClient } from "@tanstack/react-query";

const config = getDefaultConfig({
  appName: "My RainbowKit App",
  projectId: "9b4a97010a2d03a05e71209f233ab457",
  chains: [sepolia],
  ssr: true,
});

const queryClient = new QueryClient();

export const RainbowProvider = ({ children }) => {
  return (
    <RainbowKitContext.Provider value={config}>
      <WagmiProvider config={config}>
        <QueryClientProvider client={queryClient}>
          <RainbowKitProvider>{children}</RainbowKitProvider>
        </QueryClientProvider>
      </WagmiProvider>
    </RainbowKitContext.Provider>
  );
};
```

### Provider Setup

The RainbowKit provider wraps your entire application in `src/app/layout.tsx`:

```tsx src/app/layout.tsx
import { RainbowProvider } from "@/context/RainbowKitContext";

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        <RainbowProvider>
          <ThemeProvider>
            <main>{children}</main>
          </ThemeProvider>
        </RainbowProvider>
      </body>
    </html>
  );
}
```

## Using the Connect Button

RainbowKit provides a pre-built `ConnectButton` component that handles the entire wallet connection flow:

```jsx src/app/portfolio/page.js
import { ConnectButton } from "@rainbow-me/rainbowkit";

export default function PortfolioPage() {
  return (
    <header>
      <div className="container flex h-14 items-center">
        <div className="justify-center w-full hidden md:flex">
          <ConnectButton />
        </div>
      </div>
    </header>
  );
}
```

The `ConnectButton` automatically displays:
- "Connect Wallet" when disconnected
- Account address and balance when connected
- Network switcher if on wrong network
- Account modal with disconnect option

## Accessing Wallet Information

Use Wagmi hooks to access wallet information in your components:

```javascript
import { useAccount } from "wagmi";

export default function MyComponent() {
  const { address, isConnected } = useAccount();
  
  if (!isConnected) {
    return <div>Please connect your wallet</div>;
  }
  
  return <div>Connected: {address}</div>;
}
```

## Customization

You can customize the RainbowKit appearance to match your theme:

```javascript
import { darkTheme } from '@rainbow-me/rainbowkit';

<RainbowKitProvider theme={darkTheme()}>
  {children}
</RainbowKitProvider>
```

<Note>
  The current implementation uses the default RainbowKit theme, which automatically adapts to light/dark mode.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Wallet not connecting">
    - Ensure you have a Web3 wallet installed (MetaMask, Coinbase Wallet, etc.)
    - Check that your wallet is unlocked
    - Try refreshing the page
  </Accordion>
  
  <Accordion title="Wrong network">
    - RainbowKit will prompt you to switch to Sepolia
    - Click the network switcher in the connect button
    - Manually add Sepolia network to your wallet if needed
  </Accordion>
  
  <Accordion title="Transaction failing">
    - Ensure you have enough Sepolia ETH for gas
    - Check that you're connected to the correct account
    - Verify the contract address is correct
  </Accordion>
</AccordionGroup>
