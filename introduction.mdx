## Your Professional Journey, On-Chain

Onchain Portfolio is a decentralized application that allows you to create, store, and showcase your professional experience directly on the blockchain. Built on Ethereum Sepolia testnet, it provides an immutable, verifiable record of your work history and projects.

## Key Features

<AccordionGroup>
  <Accordion icon="wallet" title="Web3 Wallet Integration">
    Connect your wallet using RainbowKit with support for multiple wallet providers. Built on wagmi v2 and viem for robust Ethereum interactions.
  </Accordion>

  <Accordion icon="briefcase" title="Immutable Work History">
    Store your professional experience on-chain with the `addWork` function. Each entry includes your role, company, description, link, and timestamp.
  </Accordion>

  <Accordion icon="eye" title="Beautiful Portfolio Display">
    Showcase your experience with a modern, gradient-themed UI built with Next.js 14, React, and Tailwind CSS.
  </Accordion>

  <Accordion icon="shield" title="Decentralized & Verifiable">
    Your portfolio data is stored on Ethereum Sepolia testnet, providing transparency and immutability.
  </Accordion>
</AccordionGroup>

## Smart Contract

The Portfolio smart contract is deployed on Ethereum Sepolia testnet:

<Note>
  **Contract Address:** `0x96127ffA4375Aee0acF87443d508f2F2bd3f4620`
  
  **Network:** Sepolia Testnet
</Note>

### Core Functions

```solidity
// Add work experience to your portfolio
function addWork(
  string _role,
  string _description, 
  string _link,
  string _company
) public

// Retrieve all work experiences
function getAllWork() public view returns (ProofOfWork[] memory)
```
