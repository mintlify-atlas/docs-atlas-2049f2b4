## Get Started Fast

This guide will walk you through connecting your wallet, adding your first work experience, and viewing your on-chain portfolio.

<Note>
  **Prerequisites:** You'll need a Web3 wallet (MetaMask, Coinbase Wallet, etc.) with Sepolia testnet ETH. Get free Sepolia ETH from a [faucet](https://sepoliafaucet.com/).
</Note>

## Quick Setup

<Steps>
  <Step title="Clone and Install">
    Clone the repository and install dependencies:
    
    ```bash
    git clone <repository-url>
    cd onchain-portfolio
    npm install
    ```
    
    Start the development server:
    
    ```bash
    npm run dev
    ```
    
    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>

  <Step title="Connect Your Wallet">
    Click the **Connect Wallet** button in the navigation bar. RainbowKit will display available wallet options.
    
    <Frame>
      <img src="/images/connect-wallet.png" alt="Connect wallet with RainbowKit" />
    </Frame>
    
    Select your wallet and approve the connection. Make sure you're connected to **Sepolia Testnet**.
    
    <Warning>
      If you don't see Sepolia in your wallet, you'll need to add it manually in your wallet settings.
    </Warning>
  </Step>

  <Step title="Navigate to Add Experience">
    Once connected, click the **Add Experience** button in the header or navigate to `/form/experience`.
    
    You'll see a form with fields for your work experience:
    - **Company** - Organization name
    - **Link** - URL to project or company
    - **Position** - Your role or title
    - **Description** - Details about your work
  </Step>

  <Step title="Fill in Your Experience">
    Enter your professional experience details:
    
    ```text Example Entry
    Company: Bhopal DAO
    Link: https://x.com/Bhopal_Dao
    Position: Blockchain Developer
    Description: Developed decentralized portfolio application using Solidity, Next.js, and Web3 technologies. Implemented smart contract for immutable work history storage.
    ```
  </Step>

  <Step title="Submit to Blockchain">
    Click **Add Experience** to submit your entry to the smart contract.
    
    Your wallet will prompt you to:
    1. Review the transaction details
    2. Confirm the gas fee
    3. Sign and submit the transaction
    
    <Info>
      The transaction calls the `addWork` function on contract `0x96127ffA4375Aee0acF87443d508f2F2bd3f4620`
    </Info>
    
    Wait for the transaction to be confirmed on Sepolia testnet (usually 15-30 seconds).
  </Step>

  <Step title="View Your Portfolio">
    Navigate to `/portfolio` or click **View Portfolio** to see your on-chain experience.
    
    <Frame>
      <img src="/images/portfolio-view.png" alt="Portfolio page with gradient theme" />
    </Frame>
    
    Your experience card will display with:
    - Purple-to-pink gradient styling
    - Your role as the heading
    - Company badge
    - Description text
    - Link to your project
  </Step>
</Steps>

    // Form UI...
  );

import { useAccount, useReadContract } from "wagmi";

## Smart Contract Interaction

The `addWork` function stores your experience on-chain:

<CodeGroup>

```solidity Contract Function
function addWork(
    string memory _role,
    string memory _description,
    string memory _link,
    string memory _company
) public {
    ProofOfWork memory newWork = ProofOfWork({
        role: _role,
        description: _description,
        link: _link,
        company: _company,
        timestamp: block.timestamp
    });
    
    portfolio.push(newWork);
    emit WorkAdded(_role, _company, block.timestamp);
}
```

```typescript ABI (src/app/contract/abi.json)
{
  "inputs": [
    { "internalType": "string", "name": "_role", "type": "string" },
    { "internalType": "string", "name": "_description", "type": "string" },
    { "internalType": "string", "name": "_link", "type": "string" },
    { "internalType": "string", "name": "_company", "type": "string" }
  ],
  "name": "addWork",
  "outputs": [],
  "stateMutability": "nonpayable",
  "type": "function"
}
```

</CodeGroup>

## Configuration

The app is configured to work with Sepolia testnet via RainbowKit:

```typescript RainbowKit Configuration (src/context/RainbowKitContext.js)
import { getDefaultConfig, RainbowKitProvider } from "@rainbow-me/rainbowkit";
import { WagmiProvider } from "wagmi";
import { sepolia } from "wagmi/chains";
import { QueryClientProvider, QueryClient } from "@tanstack/react-query";

const config = getDefaultConfig({
  appName: "My RainbowKit App",
  projectId: "9b4a97010a2d03a05e71209f233ab457",
  chains: [sepolia],
  ssr: true,
});

const queryClient = new QueryClient();

export const RainbowProvider = ({ children }) => {
  return (
    <WagmiProvider config={config}>
      <QueryClientProvider client={queryClient}>
        <RainbowKitProvider>{children}</RainbowKitProvider>
      </QueryClientProvider>
    </WagmiProvider>
  );
};
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Wallet not connecting">
    - Make sure you're using a supported wallet (MetaMask, Coinbase Wallet, etc.)
    - Check that your wallet extension is installed and unlocked
    - Try refreshing the page
  </Accordion>

  <Accordion title="Transaction failing">
    - Ensure you have enough Sepolia ETH for gas fees
    - Verify you're connected to Sepolia testnet, not Ethereum mainnet
    - Check that all form fields are filled out
  </Accordion>

  <Accordion title="Portfolio not displaying">
    - Wait for transaction confirmation (check on [Sepolia Etherscan](https://sepolia.etherscan.io/))
    - Refresh the portfolio page after transaction confirms
    - Check browser console for any errors
  </Accordion>

  <Accordion title="Need Sepolia ETH">
    Get free testnet ETH from these faucets:
    - [Alchemy Sepolia Faucet](https://sepoliafaucet.com/)
    - [Infura Sepolia Faucet](https://www.infura.io/faucet/sepolia)
    - [QuickNode Faucet](https://faucet.quicknode.com/ethereum/sepolia)
  </Accordion>
